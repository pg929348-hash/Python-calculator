# Python Calculator 🧮

## calculator.py
Built from scratch using nested `if/else`. No libraries.

**What it does:**
- Takes 2 numbers from user
- Operators: `+`, `-`, `*`, `/`
- Nested `if` for divide by zero protection
- Handles invalid operators
- Works with decimals using `float()`
